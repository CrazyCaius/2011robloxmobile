# Retro Roblox 2011 Mobile Game Project

## Overview
The Retro Roblox 2011 mobile game project is a nostalgic throwback to the Roblox experience as it was in 2011. This project aims to capture the essence of the original game, focusing on gameplay mechanics, graphics, and community interaction that made Roblox popular during that time.

## Features
- **Classic Gameplay**: Experience the original gameplay mechanics that made Roblox a favorite among players.
- **Retro Graphics**: Enjoy the pixelated graphics reminiscent of early 2010s mobile gaming.
- **Community Interaction**: Engage with other players through in-game chat and collaborative gameplay.

## Setting Up the Project
1. **Clone the Repository**: Use `git clone https://github.com/CrazyCaius/2011robloxmobile.git` to clone the project to your local machine.
2. **Install Dependencies**: Make sure to install all necessary dependencies listed in the requirements file.
3. **Run the Game**: Start the game by executing the main application file. Follow instructions on how to navigate the menu settings.
